Acid-Base & Equilibrium

1.1 นิยามของกรดและเบส

กรด = รสเปรี้ยว (น้ำส้มสายชู, มะนาว)  |  เบส = รสขม ลื่น (ยาลดกรด, NaOH)

ทฤษฎีกรด (Acid)เบส (Base)ข้อจำกัด
Arrhenius
1884 | Nobel Prize 1903		 ให้ H⁺ ในน้ำ
เช่น HCl → H⁺ + Cl⁻		 ให้ OH⁻ ในน้ำ
เช่น NaOH → Na⁺ + OH⁻		 ใช้ได้เฉพาะในน้ำ ไม่อธิบาย NH₃ ได้
Brønsted-Lowry
1923		 ผู้ให้โปรตอน H⁺
(Proton donor)		 ผู้รับโปรตอน H⁺
(Proton acceptor)		 กว้างกว่า ไม่จำกัดตัวทำละลาย
Lewis รับคู่อิเล็กตรอน
(e⁻ acceptor)		 ให้คู่อิเล็กตรอน
(e⁻ donor)		 กว้างที่สุด รวม BF₃, AlCl₃

1.4 คุณสมบัติกรด-เบสของเกลือ (Salt Hydrolysis)

ไอออนของเกลือที่มาจากกรดอ่อนหรือเบสอ่อน จะทำปฏิกิริยากับน้ำ (Hydrolysis) ทำให้ pH เปลี่ยน

ที่มาของเกลือตัวอย่างpHปฏิกิริยา
กรดแก่ + เบสแก่ NaCl, KNO₃, CaCl₂ กลาง pH = 7 ไม่เกิด hydrolysis
กรดแก่ + เบสอ่อน NH₄Cl, NH₄NO₃
AlCl₃, ZnCl₂		 กรด pH < 7 NH₄⁺ + H₂O ⇌ NH₃ + H₃O⁺
Kh = Kw/Kb
กรดอ่อน + เบสแก่ CH₃COONa, NaF
Na₂CO₃		 เบส pH > 7 CH₃COO⁻ + H₂O ⇌ CH₃COOH + OH⁻
Kh = Kw/Ka
กรดอ่อน + เบสอ่อน CH₃COONH₄ ขึ้นกับ Ka vs Kb Ka > Kb → กรด
Kb > Ka → เบส
Ka ≈ Kb → กลาง
%ไฮโดรไลซิส = (ความเข้มข้นที่แตกตัว / ความเข้มข้นเริ่มต้น) × 100

1.5 สารละลายบัฟเฟอร์ (Buffer Solution)

สารละลายที่รักษา pH ให้คงที่ เมื่อเติมกรดหรือเบสเล็กน้อย หรือเจือจาง

ประกอบจาก: กรดอ่อน + เกลือของกรดอ่อน  หรือ  เบสอ่อน + เกลือของเบสอ่อน

บัฟเฟอร์กรด (pH < 7)

CH₃COOH / CH₃COO⁻
HF / F⁻

ตัวอย่าง: CH₃COOH/CH₃COONa

บัฟเฟอร์เบส (pH > 7)

NH₃ / NH₄⁺
C₅H₅N / C₅H₅NH⁺

ตัวอย่าง: NH₃/NH₄Cl

หลักการทำงาน (ระบบ CH₃COOH/CH₃COO⁻)

➕ เติม H⁺ (กรด):
CH₃COO⁻ + H⁺ → CH₃COOH
(เบสในบัฟเฟอร์ดูดซับ)
➕ เติม OH⁻ (เบส):
CH₃COOH + OH⁻ → CH₃COO⁻ + H₂O
(กรดในบัฟเฟอร์ทำปฏิกิริยา)

สมการ Henderson-Hasselbalch

pH = pKa + log([A⁻]/[HA]) ← บัฟเฟอร์กรด pOH = pKb + log([BH⁺]/[B]) ← บัฟเฟอร์เบส ช่วงบัฟเฟอร์ทำงานได้: pH = pKa ± 1 | pOH = pKb ± 1
✏️ ตัวอย่าง — บัฟเฟอร์กรด
CH₃COOH 0.30 M + CH₃COO⁻ 0.60 M (Ka = 1.8×10⁻⁵) pH = −log(1.8×10⁻⁵) + log(0.60/0.30) = 4.74 + log(2) = 4.74 + 0.30 = 5.04
✏️ ตัวอย่าง — บัฟเฟอร์เบส
NH₃ 0.30 M / NH₄Cl 0.36 M (Kb = 1.8×10⁻⁵) pOH = −log(1.8×10⁻⁵) + log(0.36/0.30) = 4.74 + 0.079 = 4.83 pH = 14 − 4.83 = 9.17

1.6 อินดิเคเตอร์ (Acid-Base Indicators)

สารอินทรีย์ที่เปลี่ยนสีตาม pH ของสารละลาย ใช้ตรวจสอบกรด-เบส

อินดิเคเตอร์สีในกรดสีในเบสช่วง pH เปลี่ยนสี
กระดาษลิตมัสแดงน้ำเงิน~7
ฟีนอล์ฟทาลีนไม่มีสีชมพู-แดง8.2 – 10.0
เมทิลออเรนจ์แดงเหลือง3.1 – 4.4
เมทิลเรดแดงเหลือง4.2 – 6.3

1.7 การไทเทรต (Acid-Base Titration)

กระบวนการหาความเข้มข้นของกรด/เบสโดยให้ทำปฏิกิริยากันพอดี

  • จุดสมมูล (equivalent point) = กรด/เบสทำปฏิกิริยากันพอดี
  • จุดยุติ (end point) = อินดิเคเตอร์เปลี่ยนสี (เลยจุดสมมูลเล็กน้อย)
ประเภทการไทเทรตpH ที่จุดสมมูลอินดิเคเตอร์ที่เหมาะ
กรดแก่ + เบสแก่= 7เมทิลเรด, ฟีนอล์ฟทาลีน
กรดอ่อน + เบสแก่> 7ฟีนอล์ฟทาลีน
กรดแก่ + เบสอ่อน< 7เมทิลออเรนจ์
สูตร: n_กรด = n_เบส → C_กรด × V_กรด = C_เบส × V_เบส
✏️ ตัวอย่าง — หาความเข้มข้นของ HCl
ไทเทรต HCl 25.00 mL ด้วย NaOH 0.10 M ใช้ไป 24.50 mL C_HCl = (0.10 × 24.50) / 25.00 = 0.098 M

Electrochemistry — ความสัมพันธ์ระหว่างปฏิกิริยาเคมีและพลังงานไฟฟ้า

2.1 ปฏิกิริยารีดอกซ์ (Redox Reaction)

ออกซิเดชัน (Oxidation)

ให้อิเล็กตรอน (e⁻)
เลขออกซิเดชัน เพิ่มขึ้น
= สารรีดิวซ์ (reducing agent)

รีดักชัน (Reduction)

รับอิเล็กตรอน (e⁻)
เลขออกซิเดชัน ลดลง
= สารออกซิไดซ์ (oxidizing agent)

ตัวอย่าง: Zn + Cu²⁺

Zn(s) → Zn²⁺(aq) + 2e⁻ ← ออกซิเดชัน (Zn ให้ e⁻, เป็นแอโนด) Cu²⁺(aq) + 2e⁻ → Cu(s) ← รีดักชัน (Cu²⁺ รับ e⁻, เป็นแคโทด) ────────────────────────────── Zn(s) + Cu²⁺(aq) → Zn²⁺(aq) + Cu(s) ← ปฏิกิริยารวม

2.2 เซลล์กัลวานิก (Galvanic/Voltaic Cell)

ส่วนประกอบแอโนด (Anode, −)แคโทด (Cathode, +)
ปฏิกิริยาออกซิเดชันรีดักชัน
สารในเซลล์ Zn-CuZn → Zn²⁺ + 2e⁻Cu²⁺ + 2e⁻ → Cu
e⁻ ไหลจากแอโนด → แคโทด (ผ่านสายไฟ)
ไอออนไหลผ่านสะพานเกลือ (salt bridge)
E°cell = E°cathode − E°anode ตัวอย่าง Zn-Cu: E°cell = 0.34 V − (−0.76 V) = 1.10 V ปฏิกิริยาเกิดเองได้ ←→ E°cell เป็น บวก แผนภาพ: Zn(s) | Zn²⁺(aq) || Cu²⁺(aq) | Cu(s) (แอโนด|สารละลาย||สารละลาย|แคโทด)

สมการเนิร์นส์ (Nernst Equation)

E = E° − (0.0591/n) × log Q (ที่ 25°C) Q = [ไอออนแอโนด] / [ไอออนแคโทด] n = จำนวน e⁻ ที่ถ่ายโอน

2.3 เซลล์แบตเตอรี่

ชนิดเซลล์แห้ง (LeClanché)แอลคาไลน์ตะกั่ว (Lead Storage)
แรงดัน~1.5 V~1.5 V6 เซลล์ × 2 V = 12 V

2.4 เซลล์อิเล็กโทรไลต์ & กฎฟาราเดย์

Q = I × t (คูลอมบ์ = แอมแปร์ × วินาที) 1 F = 96,500 C = ประจุ 1 โมลอิเล็กตรอน

🔄 Substitution (การแทนที่) อิ่มตัว

แทนที่ H ด้วยหมู่อื่น

Alkane + X₂ → RX + HX (ต้องใช้แสง hv) Benzene + X₂ → ArX + HX (ต้องมี FeX₃) CH₄ + Cl₂ →(hv)→ CH₃Cl + HCl
➕ Addition (การเติม) ไม่อิ่มตัว

เพิ่มอะตอมที่ C=C หรือ C≡C

Halogenation: C=C + Br₂ → CBr-CBr (ฟอกสีโบรมีนน้ำตาล → ใส ทันที!) Hydrohalogenation: C=C + HX → C-C(X) Hydration: C=C + H₂O → alcohol
➖ Elimination (การขจัด) RX

ขจัด HX ออกจากโมเลกุล

RX + KOH(alc) → Alkene + KBr + H₂O CH₃CH₂Br + KOH → CH₂=CH₂ + HBr ใช้ KOH ใน alcohol หรือ NaOEt/EtOH
🟤 Oxidation (ออกซิเดชัน) Alkene / Alcohol

ใช้ KMnO₄ เป็นตัวออกซิไดซ์ (สีม่วงจาง = มี C=C)

KMnO₄ อ่อน + Alkene → diol (สีม่วงจาง) KMnO₄ แรง + Alkene → กรดคาร์บอกซิลิก ความว่องไว: อิ่มตัว (Alkane) <<< ไม่อิ่มตัว (Alkene, Alkyne)
